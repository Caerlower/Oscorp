# Integrations (optional / archived)

Oscorp’s **default stack** does not require anything in this folder.

- **Telegram + research + drafts** run in `backend/` with `GROQ_API_KEY` and `python -m app.telegram.run_bot`.
- The `openclaw/` subdirectory is a **legacy optional** external agent runtime — not used by the main app.

See [../docs/growth-research.md](../docs/growth-research.md) and [../docs/telegram.md](../docs/telegram.md).
